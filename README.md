# ML Data Preprocessing

This project demonstrates basic **data preprocessing techniques using Python and Pandas**.

## Dataset

House price dataset containing:

* house size
* number of bedrooms
* house age
* price

## Tasks Performed

1. Handling missing values
2. Data transformation
3. Feature engineering
4. Data visualization
5. Saving cleaned dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Output

Cleaned dataset ready for machine learning models.
